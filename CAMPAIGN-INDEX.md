## Template Content — Full Reference

### Structure per email:
1. **Badge** — Industry label
2. **Headline** — Two-line hook with one word/phrase in brand red
3. **Subhead** — 2-sentence value prop
4. **Preview Text** — Hidden preheader for inbox preview
5. **Pain Points** — 3 cards with emoji, bold title, description
6. **Features** — 3 cards with label, title, description
7. **Stats Bar** — 3 metrics on red gradient
8. **Bonus Note** — (optional) contextual tip
9. **Final CTA** — Headline + subhead + dual buttons
